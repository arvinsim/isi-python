Definition

What is Inheritance?

Inheritance is when a class is based on another class, using the same implementation (inheriting from an object or class) specifying implementation to maintain the same behavior.
Inheritance is used to indicate that one class will get most or all its features from another class

The "object" parameter in the class definition

Advanced

Overriding

We have already seen that a child class will inherit its parent's attributes and methods.

However, what happens when an attribute or a method in a child has the same name as the parent?

The correct answer is that the child class will "override" the attribute or method of the parent class

Overriding means that the child's implementation of the attribute or method will replace the implementation of the parent for all instances of that child class

Here is an example of the child overriding some of the parent's attributes


    class Parent(object):
        message = "Hello There!"
        age = 40

    class Child(Parent):
        age = 8
        gender = "male"

    c = Child()
    print c.message
    print c.age
    print c.gender

Here is an example of the child overriding a parent's methods


    class Parent(object):
        def get_hobbies(self):
            return ['yoga', 'cooking', 'photography']

    class Child(Parent):
        def get_hobbies(self):
            return ['eating', 'playing', 'pooping']

    c = Child()
    print c.get_hobbies()

It doesn't matter if the parameters are different. If a child attribute or method has the same name as the parent, it will replaced that attribute or method.


    class Parent(object):
        def receives(self):
            print "I am too old for this $hIt! - Grumpy Parent"

    class Child(Parent):
        def receives(self, item=None):
            if item == "blanket":
                print "ba ba, blankie"
            elif item == "pacifier":
                print "binkie wuv"
            else:
                print "muh muh boo boo"

    c = Child()
    c.receives("pacifier")

Extending

We already have discussed how to override attributes and methods. However, sometimes we don't want the child class to override the some of the parent's methods. We might want to reuse those methods because they can still help. In that case, we want to "extend" the class
to extend a class, we need to use the super() built-in function The super() built-in function allows us to call attributes and methods from the parent in the child class Here is the typical way the super() built-in function is used. class Child(Parent): def method(self, arg): super(Child, self).method(arg) # Child here is the child class # super(Child, self) evaluates to the parent class(which is Parent) # super(Child, self).method(arg) calls the method named "method" # in the Parent Class super(Child, self).method(arg) Here is an example of a child class calling a parent attribute class Parent(object): message = "Don't worry child, Mommy is here" class Child(Parent): def consult_parent(self): return super(Child, self).message c = Child() print c.consult_parent() Here is an example of a child class extending a parent method """ Example of a child class extending a parent method """
