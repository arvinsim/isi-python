Definition

What is Inheritance?

Inheritance is when a class is based on another class, using the same implementation (inheriting from an object or class) specifying implementation to maintain the same behavior.
Inheritance is used to indicate that one class will get most or all its features from another class

Composition vs Inheritance

Composition

"has-a" relationship

Inheritance

"is-a" relationship

Basics

The "object" parameter in the class definition

The object parameter that you pass to a Python class is actually a built-in class in Python. Basically, by passing it, it is implying that the class you are making inherits from the "object" class

How to make a class inherit another class

To make an object inherit another class, simply replace the "object" parameter with the name of the class

Quick Exercise

Define two empty classes, ParentClass and ChildClass. Make ChildClass inherit from ParentClass

Use the function isInstance() to check if ChildClass inherited from ParentClass


    cc = ChildClass()
    print isInstance(cc, ParentClass) # Should be True

Parent and Child relationship

in the previous example, class ParentClass is defined as the parent class or superclass while class ChildClass is defined as the child class or subclass

Inherited Attributes and Methods

When a class inherits from another class, it inherits all the attributes and methods of the parent class
