Boolean Types and Operators

Boolean Values

                        
    >>> True
    True
    >>> False
    False

Boolean/Logical Operators

not (Logical NOT)

                            
    >>> not True
    False
    >>> not False
    True

and (Logical AND)

                            
    >>> True and True
    True
    >>> True and False
    False
    >>> False and True
    False
    >>> False and False
    False

Tip: As long as one of the operands is False, the result will be False

or (Logical OR)

                            
    >>> True or True
    True
    >>> True or False
    True
    >>> False or True
    True
    >>> False or False
    False

Tip: As long as one of the operands is True, the result will be True

Relational Operators

== (Equal to)

                            
    >>> 100 == 100
    True
    >>> 100 == 50
    False
    >>> -100 == 100
    False

!= (Not Equal To)

                            
    >>> 100 != 100
    False
    >>> 100 != 50
    True
    >>> -100 != 100
    True

> (Greater than)

>= (Greater than or equal to)

                            
    >>> 5 > 10
    False
    >>> 10 > 5
    True
    >>> 5 > 5
    False
    >>> 5 >= 5
    True

< (Less than)

<= (Less than or equal to)

                             
    >>> 5 < 10
    True
    >>> 10 < 5
    False
    >>> 5 < 5
    False
    >>> 5 <= 5
    True

You can use the relational operators not just with numbers, but with any data type

                            
    >>> 5 == True
    False
    >>> 10 > (1,2,3)
    False
    >>> 7 == '7'
    False

Any nonzero number is interpreted as "True"

                            
    >>> 17 and True
    True
    >>> 17 or True
    True 
    >>> 17 and False
    False

This flexibility can be useful, but there are some subtleties to it that might be confusing. You might want to avoid it (unless you know what you are doing).

                            
    >>> 17 or False
    17
    >>> 0 and True
    0
    >>> 1 > True
    False
    >>> 17 > True
    True
