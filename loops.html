While Loop

A while-loop will keep executing the code block under it as long as a boolean expression is True.

                            
    i = 0
    while i < 10:
        print i
        i += 1

A word of caution

What do you think will happen if I run this

                            
    i = 0
    while i < 10:
        print i

A word of caution

The code above would print 0 without stopping. That is because i will always be less than 10

This is what we call an infinite loop

The code would run until a system interrupt will stop it or until computing resources are exhausted
