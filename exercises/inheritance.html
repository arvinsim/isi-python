#2 Notebook Contacts

Define 4 classes

  • Address
  • Person
  • Contact
  • Notebook

Address

The Address class should contain the city and street address.
It should also contain a method show() that prints those information

Person

The Person class should contain the name and email address of a person.
It should also contain a method show() that prints those information

Contact

The Contact class inherits from both Address and Person
The Contact class should have a method show() that prints the city, street address, name and email address

Notebook

The Notebook class should have 2 methods.
A add() method where it accepts 4 parameters(city, street, name and email)
A show() method that accepts the 1 parameter(name) and prints the contact information of the person(it should use the Contact class show() method ).
If it cannot find a person what that name, it should print "Unknown "

Output

Given this code


    notes = Notebook()
    notes.add('Nancy', 'ndrew@gmail.com', 'River Heights', 'Grinell')
    notes.add('Joe', 'jhardy@gmail.com', 'Bayport', 'Terrace Road')

    notes.show('Nancy')
    notes.show('SpongeBob')

The output should be

Nancy ndrew@gmail.com
River Heights
Grinell

Unknown SpongeBob

#3 The Part Timer

Employee

Create a base class named "Employee" that requires a full_name as an argument when instantiated.
Define a method name "calculate_wage" and takes an "hours" as an argument. This method would then return the calculated salary multipled by 250.00.

PartTimeEmployee

Create a class "PartTimeEmployee" that inherits from Employee.

Give this derived class a method called "calculate_wage" that override the base class (Employee) and will take "hours" as argument as well.

It should return the part time employee's number of hours worked multiplied by 125.00

Lastly, create a method "full_time_wage" (doesn't take any arguments) and will return the employee's wage salary with hours rendered in full time. 

    emp = PartTimeEmployee("Rvin")
    print emp.calculate_wage(10) # RVin salary: 1250.00
    print emp.full_time_wage()   # RVin salary (full-time): 2500.00
