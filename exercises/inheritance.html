  • Cylinder

#2 Notebook Contacts

Define 4 classes

  • Address
  • Person
  • Contact
  • Notebook

Address

The Address class should contain the city and street address.
It should also contain a method show() that prints those information

Person

The Person class should contain the name and email address of a person.
It should also contain a method show() that prints those information

Contact

The Contact class inherits from both Address and Person
The Contact class should have a method show() that prints the city, street address, name and email address

Notebook

The Notebook class should have 2 methods.
A add() method where it accepts 4 parameters(city, street, name and email)
A show() method that accepts the 1 parameter(name) and prints the contact information of the person(it should use the Contact class show() method ).
If it cannot find a person what that name, it should print "Unknown "

Output

Given this code


    notes = Notebook()
    notes.add('Nancy', 'ndrew@gmail.com', 'River Heights', 'Grinell')
    notes.add('Joe', 'jhardy@gmail.com', 'Bayport', 'Terrace Road')

    notes.show('Nancy')
    notes.show('SpongeBob')

The output should be

Nancy ndrew@gmail.com
River Heights
Grinell

Unknown SpongeBob
